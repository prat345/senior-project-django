Autonomous Vehicle testing Dashboard

Features
  • Interactive dashboard summarizing all vehicle test drive data from database
  • Individual test dashboard displaying incident occurences respect to vehicle parameters (e.g acceleration, brake input, drive mode etc.) and the map of the location. User can get each test data by downloading CSV file
  • Query incident videos which are stored on drive. Incidents are catagorized with labels related to the cause and are divided into subcategories. User can select multiple labels with and/or condition. Allow *RUD operations on labels
  • Test log form for recording additional information about incident during vehicle testing. Allow CRUD operations on logs
  • Responsive web design and provide pagination
Tools
  • Database: MongoDB Atlas (Pymongo & Djongo)
  • Framework: Django
  • Visualizations: Ploty
  • HTML, CSS, Bootstrap
